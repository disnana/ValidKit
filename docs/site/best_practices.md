# ベストプラクティス

## スキーマを再利用する

同じ構造を何度も検証する場合、スキーマはモジュール定数として定義します。ホットパスでは `compile()` を使います。

## 入力境界で検証する

API、CLI、設定ファイル、外部イベントなど、信頼境界に入った直後に検証します。

## 例外メッセージに秘密値を出さない

```python
schema = {
    "api_key": v.str().secret(),
    "password": v.str().secret(),
}
```

## 複雑な条件は名前付き関数へ

`lambda` は短い条件に留め、複雑な `.when()` や `.custom()` は名前付き関数にするとテストしやすくなります。
